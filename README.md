# Effective Data Visualization in R <img src="images/r_logo.png" align="right" height="139"/>

This workshop covers the basics of data visualization in R using `ggplot`, `plotly`, and `leaflet`. Every workshop in the series concludes with hands-on exercises to help you put your new skills into practice.

The content for this workshop is developed and taught by the OIT Research Technology Services team at SMU in collaboration with SMU Libraries.
